# Dog Dream

Game created during **Game Jam Nitra 2026**.

## About the game
Dog Dream is a short atmospheric game made in Unity.  
The project was developed during a game jam in a limited time, focusing on unusual mechanics, mood, and visual style.

In the game you explore environments, interact with objects, and experience a surreal sequence of events from the perspective of a character and a dog.

## Play the game
The game is already built and available on itch.io:

https://itch.io/jam/game-jam-nitra-2026/rate/4266832

## Development
The project was developed in **Unity**.

Main features used:
- Character controllers
- Interactions and triggers
- Scene transitions and teleportation
- Visual effects and UI
- Low-poly assets and environments
